Help & Guide

Database

Add all JHS 3 students here first. Each student needs a name, gender, and index number. Once added, student names automatically appear in all Mock tally sheets. You can edit or delete students anytime.

Tally Sheets

Select a Mock (1–15) and a Subject, then enter each student's raw score (0–100). Grades, positions, and remarks are calculated automatically. French and Arabic are optional — only enter scores for students taking them. Click Save Scores after each subject to store data.

Master Sheet

View all subjects at once for any Mock. Toggle between Raw Scores and Grades & Aggregate views. Raw Score = Core 4 (Maths, Science, English, Social) + best 2 other subjects. Aggregate = sum of grades for Core 4 + 2 smallest grades from others. Lower aggregate = better.

Reports

Generate individual student report cards. Click View to see a student's full report. Click Print Report to print a single report on A4. Use Print All Reports to batch-print every student's report.

Analysis

See class-wide statistics for any Mock: total students, class average, best/worst scores, subject-by-subject breakdown (highest, lowest, average, pass rate), grade distribution chart, and top/bottom performers.

Settings

Update school email, phone, academic year, and upload a school logo. Toggle optional subjects (French/Arabic). Use Backup Data to export all data as a JSON file, and Restore Backup to import it back. Clear All Data wipes everything permanently.

Grade Key

1 = 90-100 (Highest)   2 = 80-89 (Higher)   3 = 70-79 (High)
4 = 60-69 (High Avg)   5 = 55-59 (Average)   6 = 50-54 (Low Avg)
7 = 40-49 (Low)   8 = 35-39 (Lower)   9 = Below 35 (Lowest)

Tip: All data is saved automatically in your browser's local storage. Remember to use Backup Data regularly to avoid losing data if you clear your browser cache.
